Introduction to Data Sturctures and Algorithms :-
  • Data :Any value or set of value.
  • Structure : Arrangement of and relation between these data.
  • Data may be organize in different ways and the logical or mathematical model of a particular organisation of data is called a data structure.
  • In simple terms, a "data structure is a particular way of organizing data within a computer so that it can be used effectively."
