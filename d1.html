    What is an algorithm

  • Any well-defined computational procedure that takes in input in that form of some value or set of value and produces an output in the form of some value, or set of values is known as an algorithm.
  • Formerly defining, an algorithm is a sequence of computational steps that transforms the input into the output.

    Correctness and Characteristics of algorithm

  • An algorithm is said to be correct if, for every input instance, it halts with the correct output. We say that a correct algorithm solves the given computational problem.
  • An incorrect algorithm might not halt all on some input instances, or it might halt with an incorrect answer.

  • Basic input and output in c++

  • Input

    If bytes flow from a device like printer,display scree, or a network connection,etc to main memory,this is called an input operation.

  • Output

    It bytes flow from the main memory to a device like a printer,display screen,or network connection,etc, this is called an output operation.

  • Stream : Sequence of bytes

    Types of streams

    Input streams : This is a type of stream where the bytes flow from a device such as a keyboard to main memory. Output Stream : This is a type of stream where the bytes flow in the opposite direction, that is, from main memory then to the device such as display screen.

  • C++ Header flies for Input/Output

    C++ provides three libraries that come with functions for performing basic input/output tasks.They include :

    • iostream : It's an acronym for standard input/output stream.This header file comes with definitions for objects like cin/cout/cerr.
    • imainp : It's an acronym for input/output mainpulators.The library comes with functions that can be used for the manipulation of streams.It contains definitions for objects like setw,setprecision,and others.
    • Fstream : This is a header file for describing the file stream.It handles data that is read from files as input or that is written to a file, the output.
    The keywords use for input and output :-
    • cout : Standard input stream used to print outputs.used with insertion operator '<<' and within double code we enclose the print text.
    • cin : Standard input stream used to take inputs.It is used with extraction operator '>>'.
    • cerr : It is a unbuffered standard error stream.It is usedto output the errors.used to display error message immediately.stores in temprary buffer(memory). Directly display on screen
    • clog : Buffered Standarderror Stream.Used to display error.similary to cerr but stores the buffer and display the error.
