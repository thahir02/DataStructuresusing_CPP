Introduction to Data Sturctures and Algorithms :-

  • Data :Any value or set of value.
  • Structure : Arrangement of and relation between these data.
  • Data may be organize in different ways and the logical or mathematical model of a particular organisation of data is called a data structure.
  • In simple terms, a data structure is a particular way of organizing data within a computer so that it can be used effectively.

      What is an algorithm

    • Any well-defined computational procedure that takes in input in that form of some value or set of value and produces an output in the form of some value, or set of values is known as an algorithm.
    • Formerly defining, an algorithm is a sequence of computational steps that transforms the input into the output.

      Correctness and Characteristics of algorithm

    • An algorithm is said to be correct if, for every input instance, it halts with the correct output. We say that a correct algorithm solves the given computational problem.
    • An incorrect algorithm might not halt all on some input instances, or it might halt with an incorrect answer.
